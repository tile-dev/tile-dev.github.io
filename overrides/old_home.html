What makes Tile special?

More than just another realtime framework, Tile (powered by Datapane) is a comprehensive SDK for creating flexible data products for your end-users in seconds.

Static generation

Tile shuns the idea that any app interaction requires a running backend server. Instead, Tile render your app to an HTML bundle which you can share in seconds through email, GitHub pages, or S3.

Python API for Javascript components

Say goodbye to writing HTML divs, and choose from a rich set of composable, interactive components. DataTables, dropdowns, tabs, and selects are accessible in a single line of Python.

Serverless backend functions

Want to add interactive forms and run backend code? Tile allows you to magically deploy Python functions and Jupyter Notebooks which you can use in your apps.

Bring Your Own Environment

Tile's ethos is to fit into where you work. Use Tile inside of Spark, Airflow, or Jupyter. Add plots and data from the libraries you love, like Pandas, Plotly, and Altair.

Components make it easy to create beautiful UIs.

Create interactive reports with dropdowns, interactive tables, and selects -- without writing a line of HTML or CSS.

app/notebook.ipynb 
import tile as t

t.View(
t.Plot(sales_v_profit, name='sale_v_profit'),
t.Form(name='end-user-params')
t.DataTable(f_data, name='forecast_data'),
t.Plot(f_plot, name='forecast_plot')
).Backend(
t.Function(name='end-users-params', path='notebook.ipynb')
).run_app()
Functions add backend code to your apps.

Want end-users to dynamically run backend scripts and notebooks? Add functions to your reports which return dynamic views.
